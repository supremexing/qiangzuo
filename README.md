# 来吧助手 LibAux | 我去图书馆服务

 
## 首次使用
1. 下载「噗噗Flag」应用，先微信登录，然后在该应用的设置中绑定手机号。
2. 在「来吧助手LibAux」应用中直接验证码登录(推荐)或手机号密码注册。   
3. 本应用在验证码登录或注册后会自动创建提供图书馆服务的来吧账号。     

 
## 应用功能
- [x] 短信登录，同步常用座位  
- [x] 创建当日的定时预约任务  
- [x] 创建明日预约及退座任务   
- [x] 增加明日预约的专用座位  
- [x] 增加当日预约的常用座位  
- [ ] 各学校软件预约冲突提醒  
- [ ] 增加场馆座位的空闲扫描
